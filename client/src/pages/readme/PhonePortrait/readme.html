Read Me
This app demonstrates how to set up the camera component to take photos from the device.
Cordova Plug-in
•
This feature requires the Cordova Camera plug-in this app
Run The App
1.
Click the Preview button to preview the app in the emulator.
2.
Tap the Take Photo component.
3.
This will bring up the ability to take a photo using the Camera or add from Library.
4.
Once you have an image selected, click the Upload button.
